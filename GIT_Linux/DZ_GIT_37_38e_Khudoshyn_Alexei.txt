(1) Что делает команда git revert и пример ее использования.
	git revert представляет собой типичную команду отмены. Однако принцип ее действия отличается от привычной отмены изменений.
	Вместо удаления коммита из истории проекта эта команда отменяет внесенные в нем изменения и добавляет новый коммит с полученным содержимым
	В результате история в Git не теряется, что важно для обеспечения целостной истории версий и надежной совместной работы.
		Пример:
			$ git log

			commit 7cc545db94bb1c6cf02897f0beabe851aa68f61b (HEAD -> master)
			Author: Oleksiy Khudoshyn <hudoshin7605@gmail.com>
			Date:   Wed Sep 7 19:50:34 2022 +0200
  			Залил вопросы
       
			commit 7daac4692754820598a055c82b564bb8ff5b4c82
			Author: Oleksiy Khudoshyn <hudoshin7605@gmail.com>
			Date:   Wed Sep 7 18:51:17 2022 +0200
			New rep! Hura!
			
			git revert HEAD (HEAD в моем случае, либо хэш коммита 7cc545db94bb1c6cf02897f0beabe851aa68f61b)
			
		команда откатит изменения до коммита 7cc545db94bb1c6cf02897f0beabe851aa68f61b New rep! Hura!, 
		но при этом не затрет историю коммитов, а создаст новый коммит.



(2) Какой командой можно изменить последний коммит?
	git commit --amend для изменения последнего комментария

(3) Что делает команда git remote rm origin?
	git remote rm удаляет подключение к удаленному репозиторию, обозначенному с помощью параметра ＜origin＞
(4) В каких случаях происходит MERGE CONFLICT?
	Конфликты возникают, когда два разраба изменяют одни и те же строки в файле или один разработчик удаляет файл, 
	который в это время изменяет другой разработчик
	
(5) Как посмотреть отличия между ветками?
	git diff ветка1 ветка2
	
(6) Какой командой удалить локальную и удаленную ветки?
	git branch -d <branchname>  локальные
	git branch -d -r <branchname> удаленные 
	
(7) Как сделать файл неотслеживаемым (untracked)?
	git clean -n, это покажет файлы, подлежащие удалению. 
	Затем выполнить само удаление командой git clean -f.
